# Project using MERN stack

Technology used: 
  
  - React
  - Material UI
  - Node.js
  - Express
  - Mongoose
  - MongoDB

The app has the following features:

### Front-end

  - List view / Card disply
  - Details view

  - Login page
  - Registration page
  - User profile page
